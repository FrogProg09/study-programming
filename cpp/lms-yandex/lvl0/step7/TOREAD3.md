# Уроки вырезания и игра в прятки с подстроками

У любой строки, кроме пустой, есть подстроки. Их можно вырезать и искать.

Вырезать подстроку — это получить все символы, начиная с заданного, в количестве N штук:

```cpp
#include <iostream>
#include <string>
#include <locale>

using namespace std;

int main() {
    setlocale(LC_ALL, ""); // Устанавливаем локаль для работы с русскими символами

    wstring phrase = L"Лучше гор могут быть только горы, на которых ещё не бывал";
    wcout << L"Подстрока  c 21 символа, длиной 11: " << phrase.substr(21, 11) << endl;
    wcout << L"Подстрока `горы` начинается на символе с индексом: " << phrase.find(L"горы")<< endl;
    wcout << L"Подстрока `Высоцкий` начинается на символе с индексом: " << phrase.find(L"Высоцкий")<< endl;

    return 0;
}
```

Что увидим на экране:

Подстрока  c 21 символа, длиной 11: только горы
Подстрока `горы` начинается на символе с индексом: 28
Подстрока `Высоцкий` начинается на символе с индексом: 18446744073709551615
Окей, с подстрокой (выделенной с помощью функции .substr(*начальный символ*,*длина*)) примерно понятно, с индексом подстроки «горы» (найденной через .find(L"горы")) — тоже. Но что это за непонятное гигантское число в последней строке?

Дело в том, что C++, если подстрока не находится, возвращает ооочень большое число — встроенную константу string::npos. Забавно, что сама константа string::npos объявлена как −1, но из-за того, что длина строки не может быть меньше нуля, сходит с ума и становится, наоборот, самой большой длиной строки, которая может быть.
