# От "Hello, world!" до "Эйяфьядлайёкюдль"

Чтобы писать красивые строки, не обязательно быть поэтом — это знает каждый программист. Ведь в конце концов строка — это всего лишь последовательность символов. С последовательностями мы пока работать не умеем (научимся чуть позже), так что будем вести себя со строками, как со строками 🚀

*Если представить буквы, как сосиски, то строка — это упаковка сосисок (Возможно, мне просто пора перекусить.)*

Чтобы задать строку (англ. string) в C++, достаточно заключить текст в двойные кавычки. Звучит знакомо, не правда ли?

Да, начиная с самого первого "Hello, world!", мы уже работаем со строками. Так что нужно просто копнуть чуть глубже и разобраться, что с ними ещё можно делать.

Сначала по доброй традиции подключим нужную библиотеку. Как и у большинства прочих библиотек языка, название у неё прямолинейное — <string>. Первый пример будет немного ностальгическим:

```cpp
#include <iostream>
#include <string> // <- помогает работать со строками

using namespace std;

int main() {
    string helloStr = "Hello, world!";
    cout << helloStr << endl;
    return 0;
}
```

Соскучились? Строковый Hello, world! снова на экране! От нашей первой программы код отличается тем, что перед выводом "Hello, world!" мы положили это значение в коробочку-переменную helloStr.

Итак, мы поздоровались. Следующим логичным шагом будет представиться. Напишем программу, которая считывает имя пользователя и приветствует его:

```cpp
#include <iostream>
#include <string>

using namespace std;

int main() {
    string name = ""; // Не забываем инициализацию
    cout << "Привет, как тебя зовут?" << endl;
    cin >> name; // Запоминаем, забывать имя сразу же после знакомства невежливо
    cout << "Приятно познакомиться, " << name << "!" << endl;
    return 0;
}
```

приветствие. Но что будет, если пользователь крайне официален и представится по имени и фамилии, например, "Иван Альпинистов"?

Привет, как тебя зовут?
Иван Альпинистов
Приятно познакомиться, Иван!
Что это за панибратство? Где фамилия?

Когда мы используем оператор >>, он считывает данные только до первого встреченного пробела — как если бы из всего заказа в онлайн-магазине вам доставили только первую позицию.

Что же делать? Есть два пути: сделать две переменные и загружать в них данные по отдельности. Для случая с именем и фамилией такое вполне оправдано, потому что скорее всего нам захочется запомнить их отдельно (например, для поиска тёзок и однофамильцев):

```cpp
#include <iostream>
#include <string>

using namespace std;

int main() {
    string name = "", lastName = "";
    cout << "Привет, как тебя зовут?" << endl;
    cin >> name >> lastName;
    cout << "Приятно познакомиться, " << name << " " << lastName << "!" << endl;
    return 0;
}
```

Но что, если мы не знаем, сколько слов введёт пользователь? Если хочется большего, нужно использовать функцию getline, которая в качестве первого параметра принимает cin, а в качестве второго — переменную, в которую мы хотим записать значение. Работает, как пылесос, для всего ввода, включая пробелы (и тайные мысли пользователя):

```cpp
#include <iostream>
#include <string>

using namespace std;

int main() {
    string fullName = "";
    cout << "Привет, как тебя зовут?" << endl;
    getline(cin,fullName);
    cout << "Приятно познакомиться, " << fullName << "!" << endl;
    return 0;
}
```
