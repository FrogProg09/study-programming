# И снова привет!

Ограничение времени     1 секунда
Ограничение памяти      64 Мб
Ввод                    стандартный ввод
Вывод                   стандартный вывод

Вы впервые активируете свой новый персональный AI-ассистент. Для создания вашего профиля он просит вас представиться, указав полное имя (имя и фамилию).

После этого AI не просто приветствует вас, но и генерирует для вас уникальный личный позывной для дальнейшего взаимодействия. Это доказывает, что он корректно обработал и сохранил ваши данные.

Правила генерации позывного: Позывной создается из трех частей, соединенных дефисами:

- Первая буква вашего полного имени.
- Последняя буква вашего полного имени.
- Общая длина вашего полного имени (включая пробел).

Напишите программу, которая считывает полное имя пользователя и выводит приветствие вместе с сгенерированным позывным.

## Формат ввода

Программа запрашивает одну строку, которая может содержать пробелы — полное имя пользователя.

## Формат вывода

Программа должна вывести трехстрочное сообщение:

Активация... Профиль создан.
Добро пожаловать, [Полное Имя]!
Ваш личный позывной: [Позывной]
