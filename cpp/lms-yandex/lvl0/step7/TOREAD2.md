# Склеиваем строки и катаемся по индексам

Так как строки — более сложный тип данных, чем числа, то и вариантов взаимодействия с ними значительно больше. По ходу курса мы будем периодически узнавать новые функции, а пока разберёмся с базовой.

Во-первых, строки можно складывать (и это всё, что можно с ними делать с математической точки зрения):

```cpp
string a = "Hello", b = ", world", c = a + b
```

Такую операцию называют сложением, из-за использования математического знака «плюс», или конкатенацией строк.

Строки упорядочены, потому что нам важен порядок символов, чтобы понять смысл фразы. (Будь это иначе, вместо предыдущего предложения я бы написал: «п, увтемас змлдоо поурвпмжкырыо р.вач тоиорысСео мыкнчояпт сдтфбьчоаяляниоот о ,нн», — и вы бы меня тоже поняли). А значит, есть возможность получить элемент, стоящий на известной позиции, по его номеру — индексу в строке.

Допустим, мы делаем суперсекретный шифровщик, который для каждого слова берёт его первый, последний и средний символы. (Суперсекретный — потому что восстановить исходное сообщение в общем случае не получится, разве что общаться словами исключительно из трёх букв.)

```cpp
#include <iostream>
#include <string>

using namespace std;

int main() {
    string word = "", secret = "";
    int length;
    cout << "Введи слово для шифровки: ";
    cin >> word;
    length = word.length(); // Получаем длину слова
    cout << "Длина слова - " << length << endl;
    secret += word[0]; // Индексы в программировании принято начинать с нуля
    secret += word[length - 1]; // Индексы указываются в квадратных скобках  
    secret += word[length / 2]; // Не обязательно числом, можно переменной или выражением
    cout << "Зашифрованное слово: " << secret << endl;

    return 0;
}
```

Тут мы используем несколько классных особенностей строк:

вычисление длины строки: word.length()
обращение к элементам по индексам: word[0]
Введи слово для шифровки: Eyjafjallajokull
Длина слова - 16
Зашифрованное слово: Ell

**Повторимся:**

    - нумерация букв в строке идет с нуля. То есть первая буква слова будет с индексом 0
    - индекс указывается в квадратных скобках (как будто лифт едет на нужный этаж)
    - можно указывать не число напрямую, а переменную, которая его содержит, или выражение (сначала оно будет вычислено)

Но если мы попробуем работать с русскими словами, то столкнёмся с той же проблемой, что и при использовании char. Так что снова отправимся в мультивселенную юникода:

```cpp
#include <iostream>
#include <string>
#include <locale>

using namespace std;

int main() {
    setlocale(LC_ALL, ""); // Устанавливаем локаль для работы с русскими символами

    wstring word = L"", secret = L"";
    int length;

    wcout << L"Введи слово для шифровки: ";
    wcin >> word; // Читаем строку как wstring

    length = word.length();
    wcout << L"Длина слова - " << length << endl;

    // Формируем зашифрованное слово
    secret += word[0];
    secret += word[length / 2];
    secret += word[length - 1];

    wcout << L"Зашифрованное слово: " << secret << endl;

    return 0;
}
```

Что ещё можно делать со строками? Например, преобразовывать строки в числа и обратно:

```cpp
#include <iostream>
#include <string>

using namespace std;

int main() {
    string code = "";
    cout << "Введите любое число: ";
    cin >> code;
    cout << "Прибавляем символ `1`: " << code + '1' << endl;
    cout << "Преобразуем в число и добавляем 1: " << stoi(code) + 1 << endl;

    return 0;
}
```

Для преобразования строки в число используется функция stoi (сокращение от "string to integer", потому что столько букв писать не хочется), а для обратной махинации — to_string (если бы функцию придумывал я, то назвал бы её просто tos).
