# Хоккейная команда

Ограничение времени     1 секунда
Ограничение памяти      64 Мб
Ввод                    стандартный ввод
Вывод                   стандартный вывод

Вы работаете программистом в крупном спортивном новостном агентстве. Данные о результатах матчей поступают от автоматической системы в сжатом техническом формате: Город_НазваниеКоманды (например, Vegas_Golden_Knights).

Ваша задача — написать программу, которая "очеловечивает" эти данные: извлекает из строки только название команды и генерирует для неё яркую фанатскую кричалку для публикации в соцсетях.

Правила генерации кричалки:

Кричалка состоит из названия команды, повторенного дважды через восклицательный знак и пробел, с добавлением слова Go! в конце.

## Формат ввода
Программа принимает одну строку без пробелов в формате Город_НазваниеКоманды.

## Формат вывода

Программа должна вывести одну строку — сгенерированную кричалку.
