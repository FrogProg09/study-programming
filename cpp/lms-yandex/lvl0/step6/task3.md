# Сдвиг салата

Ограничение времени 1 секунда
Ограничение памяти  64 Мб
Ввод                стандартный ввод
Вывод               стандартный вывод

Во время раскопок древнего города вы нашли свиток с загадочными символами. Ваш коллега, специалист по древним языкам, предположил, что это зашифрованное послание, созданное с использованием шифра Цезаря. Шифр Цезаря — это метод шифрования, при котором каждый символ в тексте "сдвигается" на определённое количество позиций в алфавите.

Чтобы расшифровать послание, вам нужно понять, как работает этот сдвиг. Вы решаете начать с простого: написать программу, которая демонстрирует, как символы меняются при сдвиге на заданное число.

Вам поручено написать программу, которая:

    1. Запрашивает у пользователя символ (тип char), который нужно зашифровать.
    2. Запрашивает у пользователя целое число n - величину сдвига.
    3. Вычисляет новый символ, сдвинутый на n позиций вперёд в таблице ASCII.
    4. Выводит исходный символ, его ASCII-код, новый символ и его ASCII-код в следующем формате:

    ```
    Исходный символ: [символ] (код: [код])
    Новый символ: [символ] (код: [код])
    ```
    
## Формат ввода

Программа запрашивает:
    - Символ (тип char) — исходный символ.
    - Целое число n(n ∈ [1;26]) — величина сдвига.

## Формат вывода
    
Программа выводит исходный и новый символы вместе с их ASCII-кодами в формате, указанном в условии.

Пример 1

```Ввод
A
10
```

```Вывод
Исходный символ: A (код: 65)
Новый символ: K (код: 75)
```

## Примечания

Убедитесь, что новый символ остаётся в пределах печатных символов ASCII (коды от 32 до 127)
