# Уже магия регистров

Ограничение времени	1 секунда
Ограничение памяти	64 Мб
Ввод	стандартный ввод
Вывод	стандартный вывод
Вы программируете низкоуровневый драйвер для специализированного промышленного принтера. Принтер получает команды в виде прописных (заглавных) букв английского алфавита (например, P — Print, C — Calibrate). Однако его внутренний контроллер для активации различных режимов работы требует команды в виде строчных букв с дополнительным числовым смещением.

Ваша задача — написать программу-конвертер, которая преобразует входящий сигнал в корректный формат для контроллера.

Напишите программу, которая:

Принимает на вход прописную букву и целочисленное значение для смещения.
Вычисляет смещение для перевода в нижний регистр.
Преобразует входную букву в строчную, используя вычисленное смещение.
Применяет к полученной строчной букве дополнительное числовое смещение.
Выводит итоговый символ команды.
Формат ввода
В одной строке через пробел вводятся два значения: прописная буква английского алфавита C и целое число N (смещение).

Формат вывода
Программа должна вывести итоговый символ команды в формате:

Итоговый символ команды: [символ]
