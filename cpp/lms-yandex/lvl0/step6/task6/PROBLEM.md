# Символьная арефметика

Ограничение времени	1 секунда
Ограничение памяти	64 Мб
Ввод	стандартный ввод
Вывод	стандартный вывод
Вы работаете над компонентом для библиотеки визуализации данных. Одна из функций этой библиотеки — создание "символьных градиентов" для отображения данных в текстовом режиме. Градиент определяется начальным и конечным символами (например, от 'A' до 'G').

Для правильного построения градиента системе необходимы два ключевых параметра этого символьного диапазона:

Центральный символ: Символ, который находится ровно посередине между начальным и конечным.
Дистанция: Количество шагов по таблице ASCII, которое разделяет эти два символа.
Напишите программу, которая принимает на вход два символа и вычисляет для них эти два параметра.

Формат ввода
В одной строке через пробел вводятся два печатных символа ASCII.

Формат вывода
Программа должна вывести результаты анализа в формате:

Начальный символ: [символ_1]
Конечный символ: [символ_2]

---
Центральный символ: [средний_символ]
Дистанция: [дистанция]
